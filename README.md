<details>
<summary><b>⚙️ (Tuỳ chọn) Bật Contribution Snake cho tài khoản của bạn</b></summary>

1) Tạo folder <code>.github/workflows</code> trong repo <code>hansukedev</code> (repo trùng username).  
2) Thêm file <code>snake.yml</code> với nội dung sau:
    
```yaml
name: Generate snake
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@master
        with:
          github_user_name: hansukedev
          outputs: dist/snake.svg
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
3) Sau khi workflow tạo xong file <code>snake.svg</code> trên branch <code>output</code>, đổi link ở mục “Contribution Snake” thành:  
<code>https://raw.githubusercontent.com/hansukedev/hansukedev/output/snake.svg</code>
</details>
